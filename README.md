### 🎮 [Esports Market Intelligence Platform](https://github.com/YOUR_GITHUB_USERNAME/esports-market-intelligence)
> **Azure · ADF · Databricks · Delta Lake · PySpark · Power BI**

A cloud-native platform on Azure that identifies **trending games** using live data from **Riot Games, Twitch & RAWG APIs**.
- Ingests **50K+ daily events** via ADF into a Bronze/Silver/Gold Delta Lakehouse on ADLS Gen2
- Computes a proprietary **30-day Momentum Score** across 100+ games using PySpark rolling averages
- Surfaces results in a **Power BI dashboard** with top-10 rankings, viewership trends & undervalued games scatter plot
